CHANGELOG

What we've shipped recently.

Tymr is in active development with a design-partner cohort open for Q2 2026. Here's what's landed in the last few weeks — in reverse chronological order.

2026-04-19 · WEBSITE
Compliance repositioning — homepage, ROI calculator, UTEv1 page, changelog
Homepage rewritten around the R&D Credit + ASC 350-40 beachhead. Floor visual redesigned as a live audit ledger with classification chips and a § 174 credit estimate. New ROI calculator, UTEv1 spec page, and this changelog. OG/Twitter meta added everywhere.
positioning website
2026-04-17 · OPEN SOURCE
UTEv1 v0.1 + TypeScript & Python SDKs
First public draft of the Universal Time Event spec — JSON Schema 2020-12 — plus reference client SDKs in TypeScript and Python. Includes wrapClaudeAgent (Anthropic) and TymrCallbackHandler (LangChain). Generated types, buffer/flush semantics, session/task context managers.
UTEv1 SDK
2026-04-17 · INTEGRATION
GitHub org-level webhook
Push, pull-request, and review events now flow straight into the audit ledger via a single org-level webhook. Adds the third deep-integration source system (after Jira and calendars). HMAC signature verification on the roadmap.
integration github
2026-04-16 · STRATEGY
Beachhead confirmed — R&D Credit + SOX CapEx
Repositioning from horizontal workforce intelligence to the audit-ready labor-cost layer for IRS § 174 and ASC 350-40. Tax-firm channel outreach begins; direct CFO outbound on deck. Professional-services vertical parked indefinitely.
positioning
2026-04-16 · PRODUCT
Org-level visibility flags & tier repack
Floor, cost-ticker, and agent features are now optional per org. Privacy-sensitive deployments can run Tymr as a pure evidence ledger without the live-floor UX. Three new tiers mapped to visibility levels.
privacy packaging
2026-04-14 · PRODUCT
Per-org integration activity config
Per-org taxonomy, canonical override rules, and preview-before-enable flow. Admins can customize which source events map to which classification, with a diff view before committing changes.
